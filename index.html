html

(Voltar)

A tag (html) é a tag raiz. nela fica toda a pagina.

title

(Voltar)

A tag (title) é que define o titulo da pagina que vai aparecer na aba do navegador e ela fica localizada na head.

head

(Voltar)

A tag (head) podemos chamar de cabeça da pagina nela estão as metas o titulo e ainda pode lincar o CSS. ela não aparece na pagina web.

body

(Voltar)

A tag (body) podemos chamar de o corpo da pagina, nela fica a aparte visivél da pagina pelo usuario.

h1,h2,h3,h4,h5,h6

(Voltar)

A tag (h) ela é dividida em seis que vai do h1 até h6, são titulos onde o h1 é o mais importante e só deve exirtir apenas um por pagina, os demais funcionam como subtitulos e podem ser repetidos.

italico

(Voltar)

A tag (i) transforma o texto selecionado em italico

strong

(Voltar)

A tag (strong) transforma o texto selicionado em negrito

input

(Voltar)

A tag (input) é um campo quando por exemplo type="text" abre uma caixa para escrever nela.

img

(Voltar)

A tag (img) é utilisada para colocar umaimagem na pagina.

paragrafo

(Voltar)

A tag (p) é utilisada para colocar um paragrafo na pagina.

blockquote

(Voltar)

A tag (blockquote) é utilisada para colocar uma citação na pagina.

sublinhado

(Voltar)

A tag (u) é utilisada para sublinhar.

mark

(Voltar)

A tag (mark) é utilisada para marcar o texto como uma caneta marca-texto.

sup

(Voltar)

A tag (sup) é utilisada para colocar uma letra ou um caractere na parte superior da linha do texto.

listas

(Voltar)

As tags das listas compreendem varios tipos de listas as principais são as ordenadas (ol) e não ordenadas (ul), mas essas listas para serem utilizadas precisão das tag filhas (li), que são os itens das listas.

link

(Voltar)

A tag (a) se refere a link, com essa tag podemos ligar o nossa pagina a outras e também pode navegar dntro da propria pagina, ciando links dentro da propria pagina.

hr

(Voltar)

A tag (hr) cria uma linha continua horizontal na pagina servindo para estetica.

small

(Voltar)

A tag (small) é uma tag para criar um pequeno texto.

O que é inpetor de elementos?

(Voltar)

É a ferramenta do desenvolvedor que é utilizada no navegador para possa visualisar todos os detalhes da pagina, abre-se um console e nele você navega por todos os elementos da pagina e tem acesso a todas informações de como foi criado cada um.
